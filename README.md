# Employee and Department Management System

## Project Overview

This SQL project demonstrates how to design and manage a relational database for employee and department information.

The system allows storing employee details, department information and performing SQL queries for data analysis.

## Tools Used

SQL  
MySQL  

## Database Schema

The database contains two tables:

Departments  
Employees  

The Employees table is connected to Departments using a foreign key.

## SQL Operations Used

SELECT  
WHERE  
GROUP BY  
UPDATE  
DELETE
