# Employee and Department Management System

This project demonstrates how SQL can be used to manage employee and department data in an organization.

The database consists of two tables:

1. Departments
2. Employees

Features of this project:

• Database creation  
• Table design  
• Data insertion  
• Data analysis using SQL queries  
• Data update and deletion operations  

This project helps understand relational database concepts and SQL operations used in real-world business scenarios.
